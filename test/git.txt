git 分布式版本控制系统
cvs和svn 都是集中式版本控制系统

命令
   1、创建一个版本库 选择一个合适的地方创建一个空目录（仓库）
   mkdir 文件名
   2、显示当前路径
   $pwd  

    第二步、
    通过git init 命令把这个目录变成Git可以管理的仓库
    执行后 文件夹就会出现.git目录  如果没有。执行ls -ah
	
	注意事项：不要使用win自带记事本进行编辑。建议使用Notepad++
	
	添加文件到Git仓库 分两步
	第一步:使用命令 git add 文件名.后缀名
	第二步：提交 git commit -m "本次提交更新内容说明"
